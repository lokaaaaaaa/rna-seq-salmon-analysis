# RNA-seq Salmon Analysis Pipeline

End-to-end RNA-seq transcript quantification workflow using public human sequencing data.

## Tools Used
- FastQC
- MultiQC
- Salmon
- Bash
- Linux (WSL)
- Git / GitHub

## Workflow
1. Download FASTQ files
2. Quality Control with FastQC
3. Summarize with MultiQC
4. Quantify transcripts using Salmon
5. Export quant.sf results

## Example Dataset
SRR1039508

## Output
- QC reports in `/qc`
- Quantification in `/salmon_quant`
